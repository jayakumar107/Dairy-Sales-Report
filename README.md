1. Dairy Sales Report
A comprehensive data analytics dashboard built using a dairy farm operations dataset. 
2. Purpose
This dashboard provides insights into farm performance, product inventory, sales trends, stock management, and revenue generation across multiple dairy products and sales channels.
3. Tech Stack
Power BI Desktop - Main data visualization platform used for report creation

Power Query - Data transformation and cleaning layer for reshaping and preparing the data 

DAX (Data Analysis Expression) - Used for calculated measures, dynamic visuals and conditional logic 

Data Modeling - Relationships established among tables to enable cross-filtering and aggregation 

File Format - .pbix for development and .png for dashboard preview

4. Dataset
https://www.kaggle.com/datasets/suraj520/dairy-goods-sales-dataset/data

5. Features
• Revenue Analysis

• Sales Channel Insights

• Product Performance Analysis

• Iventory and Shelf Life Monitoring

• Farm Operations Analytics

• Geographic Analysis
